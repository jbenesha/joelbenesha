About Style Guide

What is a living style guide

Living style guides is a critical tools for web developement. Living style guide an essential tools to communicate design standards with developers and ordganisations. This helps maintain the web organized and distil especially when dealing with complex interfaces(Lambert, 2017).

Why is a living style guide useful to developers and designers

This is done by ensuring developers and designers are in sync. Developers working as a team in projects are now able to work efficiently and effectively whilst avoiding inconsistency. Allows developers to adapt existing patterns to new problems without starting from scratch when solving problems (Hildebrandt, 2017).
